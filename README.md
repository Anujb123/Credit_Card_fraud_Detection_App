# 💳 Credit Card Fraud Detection System

## 🚀 Overview

This project focuses on detecting fraudulent credit card transactions using Machine Learning techniques. The model analyzes transaction patterns and identifies anomalies that indicate potential fraud.

---

## 🎯 Objectives

* Build a reliable fraud detection model
* Handle imbalanced datasets effectively
* Achieve high precision and recall for fraud cases
* Provide a scalable ML pipeline for real-world use

---

## 🛠️ Tech Stack

* **Programming Language:** Python
* **Libraries:** NumPy, Pandas, Scikit-learn, Matplotlib, Seaborn
* **Model Persistence:** Joblib
* **Environment:** Jupyter Notebook

---

## ⚙️ Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/Anujb123/Credit_Card_fraud_Detection_App.git
cd Credit_Card_fraud_Detection_App
```

### 2. Create virtual environment

```bash
python -m venv venv
venv\Scripts\activate   # Windows
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```
